### **createProject(data)**

```typescript
// Method: POST
// Input: Omit<ProjectFormData, 'image_file'>
```

```typescript
ProjectFormSchema = {

  linkedin_url: URL string or empty
```

---

### **deleteProject(data)**

```typescript
// Method: DELETE
// Input: { id: string }
// Returns: { success: true }
// Side Effects: Removes image from storage

Example: await deleteProject({ data: { id: "uuid-1" } });
```

**Process**:

1. Fetches project to get image URL
2. Deletes image from storage (if exists)
3. Deletes project from database
4. Returns success

---

### **uploadProjectImage(data)**

```typescript
// Method: POST
// Input: { fileName: string; base64: string }
// Returns: { url: string }

Example: const { url } = await uploadProjectImage({
  data: {
    fileName: "1234567890-project.jpg",
    base64: "data:image/jpeg;base64,/9j/4AAQSkZJRg...",
  },
});
```

**Process**:

1. Converts base64 to buffer
2. Uploads to `projects` bucket
3. Returns public URL
4. URL ready to save to database

**Validation**:

- File size: < 5MB
- Content type: `image/*`
- Filename: `{timestamp}-{original}`

## 📋 Database Schema

### **projects table**

```sql
CREATE TABLE projects (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  title VARCHAR(255) NOT NULL,
  description TEXT NOT NULL,
  github_url VARCHAR(255),
  linkedin_url VARCHAR(255),
  skills TEXT[] NOT NULL DEFAULT ARRAY[]::TEXT[],
  image_url VARCHAR(500),
  created_at TIMESTAMP NOT NULL DEFAULT NOW(),
  updated_at TIMESTAMP NOT NULL DEFAULT NOW()
);
```

**Field Details**:

| Field        | Type         | Constraints             | Description              |
| ------------ | ------------ | ----------------------- | ------------------------ |
| id           | UUID         | PRIMARY KEY             | Auto-generated unique ID |
| title        | VARCHAR(255) | NOT NULL                | Project title            |
| description  | TEXT         | NOT NULL                | Full project description |
| github_url   | VARCHAR(255) | NULLABLE                | Repository link          |
| linkedin_url | VARCHAR(255) | NULLABLE                | LinkedIn post link       |
| skills       | TEXT[]       | NOT NULL, DEFAULT []    | Array of skill tags      |
| image_url    | VARCHAR(500) | NULLABLE                | Supabase Storage URL     |
| created_at   | TIMESTAMP    | NOT NULL, DEFAULT NOW() | Creation timestamp       |
| updated_at   | TIMESTAMP    | NOT NULL, DEFAULT NOW() | Last update timestamp    |

**Indexes**:

```sql
CREATE INDEX idx_projects_created_at ON projects(created_at DESC);
```

### **Storage Bucket: projects**

```
Bucket ID: projects
Public: true
Max File Size: 5GB (Supabase default)
```

**Typical File Path**:

```
projects/
  └── {timestamp}-{filename.ext}

Example: projects/1707932400000-portfolio.jpg
```

**Public URL Format**:

```
https://[project-id].supabase.co/storage/v1/object/public/projects/{filename}
```

2. **Update Types**:

3. **Update Validation Schema**:

```typescript

const ProjectFormSchema = z.object({
  // ... existing

```

4. **Update Form Component**:

```typescript

<input name="category" value={formData.category} onChange={handleInputChange} />
```

5. **Update Grid Display**:

```typescript
// src/components/ProjectGrid.tsx
<span className="text-sm text-slate-300">{project.category}</span>
```

### **Add Sorting/Filtering**

```typescript
// In ProjectsDashboard.tsx
const [sortBy, setSortBy] = useState("newest");

const sortedProjects = useMemo(() => {
  const sorted = [...projects];
  if (sortBy === "newest") {
    return sorted.sort(
      (a, b) =>
        new Date(b.created_at).getTime() - new Date(a.created_at).getTime(),
    );
  }
  return sorted;
}, [projects, sortBy]);
```

### **Add Search Functionality**

```typescript
const [searchTerm, setSearchTerm] = useState("");

const filteredProjects = useMemo(() => {
  return projects.filter(
    (p) =>
      p.title.toLowerCase().includes(searchTerm.toLowerCase()) ||
      p.description.toLowerCase().includes(searchTerm.toLowerCase()),
  );
}, [projects, searchTerm]);
```
